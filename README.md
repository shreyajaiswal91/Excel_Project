# Excel_Project
Bike Sales Analysis using Microsoft Excel | Data Cleaning, Dashboard Creation, Pivot Tables, Slicers and Interactive Visualizations
Project Overview

This project analyzes bike sales data using Microsoft Excel to uncover customer purchasing patterns and business insights.
The dataset was cleaned, transformed, and visualized through an interactive dashboard to help understand the factors influencing bike purchases.

Objectives
Clean and prepare raw sales data.
Analyze customer demographics and purchasing behavior.
Create interactive dashboards for business decision-making.
Identify trends based on age, income, gender, marital status, occupation, and region.

Tools Used
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
Data Cleaning Techniques
Excel Dashboard

Key Features
Data cleaning and preprocessing
Duplicate removal
Standarising the range values of salary
Age group categorization
Interactive dashboard with slicers
Pivot Table analysis
Dynamic charts and visualizations


Dashboard Insights
The dashboard provides insights into:

Bike purchases by gender
Average income per purchase
Customer age groups
Commute distance analysis
Regional sales comparison
Occupation-wise purchasing trends


Project Outcome

The analysis helps identify customer segments most likely to purchase bikes, enabling businesses to make data-driven marketing and sales decisions.
